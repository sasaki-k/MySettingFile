# MySettingFile

下記設定ファイルを公開.

- .zshrc
- .tmux.conf
- .emacs.d

## .zshrc

zsh用のコンフィグファイル

## .tmux.conf

tmux用の設定ファイル.
Sampleは, 自分の環境に合わせて変更.

## .emacs.d

emacs用の設定ファイル.
メインはinit.el.
GNU Globalにも対応しているので，インストールすれば
emacsでGlobalが使える（init.elのコメントアウト部分を外す).

また、emacs起動前に下記コマンドを実行

```sh
mkdir elpa
mkdir elisp
mkdir public_repos
```
